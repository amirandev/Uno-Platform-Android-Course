# Google Play Store

## Prerequisites

1. **Google Play Developer account** ($25 one-time fee)
2. **Signed AAB** (Android App Bundle)
3. **App icons, screenshots, description**

## Upload to Play Console

1. Go to [play.google.com/console](https://play.google.com/console)
2. Create new app
3. Fill in store listing:
   - App name, short/long description
   - Screenshots (2-8, 720p or higher)
   - Feature graphic (1024×500)
   - Icon (512×512)
   - App category

## App Content

- **Privacy Policy** — Required if app handles user data
- **Ads** — Declare if using ads
- **Rating questionnaire** — Content rating

## Release Track Options

| Track | Purpose |
|-------|---------|
| **Internal testing** | Up to 100 testers, no review |
| **Closed testing** | Invite-only, needs review |
| **Open testing** | Public testing link |
| **Production** | Public release |

## Submit for Review

Upload AAB → fill production details → review (hours to days).

## Common Rejection Reasons

- Missing privacy policy
- Crashing on startup
- Incomplete/dead features
- Insufficient testing
