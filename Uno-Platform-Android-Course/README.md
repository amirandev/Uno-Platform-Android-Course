# Uno Platform Android Development — A-to-Z Course

Build native Android apps using C# and WinUI/XAML with **Uno Platform**.

## What You'll Learn

- C# fundamentals for mobile development
- XAML & WinUI 3 UI design
- MVVM architecture & data binding
- Navigation, networking, device APIs
- Publishing to Google Play Store

## Prerequisites

- Windows PC with Visual Studio 2022+
- Basic programming knowledge (any language)
- Android device or emulator
